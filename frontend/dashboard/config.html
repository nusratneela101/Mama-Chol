📖 Platform Setup Guide

  1. Install v2rayNG from Google Play or GitHub
  2. Open app → tap ☰ menu → Subscription group settings
  3. Tap + → paste your Subscription URL above → OK
  4. Pull down to refresh → servers will appear
  5. Tap any server → press ▶ to connect
  1. Install Shadowrocket ($2.99) or Sing-box (free) from App Store
  2. Tap + top right → choose Subscribe
  3. Paste Subscription URL → tap Done
  4. Tap Update to fetch servers
  5. Toggle the main switch to connect
  1. Download v2rayN from GitHub
  2. Extract ZIP → run v2rayN.exe (requires .NET 6+)
  3. Click Servers → Add subscription → paste URL
  4. Click Update subscription
  5. Right-click tray icon → System proxy → Set system proxy
  1. Install V2Box from Mac App Store (free)
  2. Open V2Box → Subscriptions tab → tap +
  3. Paste Subscription URL → Update
  4. Select preferred server → toggle to connect
