🤖 Android Setup — v2rayNG

Free App

Step 1: Download the App

Download v2rayNG from Google Play Store or GitHub (for China users).

📥 Google Play Store
📥 GitHub (APK)

Step 2: Import Configuration

After subscribing, go to your dashboard and:

  1. Open v2rayNG → tap the + button
  2. Select "Import config from QR code"
  3. Scan the QR code from your dashboard
  4. Tap the config → press ▶ to connect

Alternative: Subscription URL

Or use the subscription URL for automatic updates:

  1. Tap + → "Import config from URL"
  2. Paste your subscription URL from dashboard
  3. Tap Update subscription
  4. Select best server and connect
💡 Subscription URL auto-updates when we add new servers — you never need to reconfigure!

🪟 Windows Setup — v2rayN

Free & Open Source

Download v2rayN

Download the latest release from GitHub. Requires Windows 10 or later.

📥 Download v2rayN

Setup Steps

  1. Extract the ZIP file and run v2rayN.exe
  2. Click Servers → Add subscription
  3. Paste your subscription URL from dashboard
  4. Click Update all subscriptions
  5. Right-click tray icon → Set system proxy

System Requirements

  • Windows 10 / 11 (64-bit)
  • .NET 6.0 or later
  • 100 MB disk space
💡 Enable TUN mode in v2rayN settings for system-wide VPN coverage (routes all app traffic).

💻 macOS Setup — V2Box

Free

V2Box is a native macOS app supporting all our protocols with a clean interface.

📥 Download from Mac App Store
  1. Open V2Box → Subscriptions tab
  2. Tap + and paste your subscription URL
  3. Tap Update to fetch servers
  4. Select a server and toggle VPN on

Alternative: Clash for macOS

Advanced users can use ClashX Pro with our Clash configuration file.

🐧 Linux Setup — v2ray-core

Free & Open Source

Install via Script

Run the official installer script:

bash <(curl -L https://raw.githubusercontent.com/v2fly/fhs-install-v2ray/master/install-release.sh)
bash <(curl -L https://raw.githubusercontent.com/v2fly/fhs-install-v2ray/master/install-release.sh)

Configure

  1. Download your config JSON from the dashboard
  2. Place it at /usr/local/etc/v2ray/config.json
  3. Run: systemctl start v2ray
  4. Set system proxy to 127.0.0.1:1080

GUI Option: Nekoray

Prefer a GUI? Nekoray supports all our protocols on Linux.
